型、値、変数

2016.07.16

はじめに

少しづつややこしくなってくるよ。型と値と変数について。
簡単な概念くらいは分かってるかと思います。変数に代入されている中身？「123」とか「Hello World」とかが値です。
プログラミング言語で操作する値の種類の事を型といいます。
でもって、値を保管？代入？しておくのが変数です。
JavaScriptの型には大きく分けて、基本型とオブジェクト型の2種類があります。

型

基本型

  • 数値 → 「123」とか「1.23」とか。
  • 文字列 → 「Hello World」とか「文字列」とか「１２３」とか。
  • 理論値 → 「true」か「false」
  • null
  • undefined

オブジェクト型

  • 上記基本型以外のもの。
  • プロパティの集合体（プロパティは名前と値を持ち、数値や文字列など基本型の値やオブジェクトを保持できます。）
  • グローバルオブジェクト
  • 配列
  • 関数
  • クラス

クラスはかなりややこしいですよ。
クラスとは、コンストラクタで初期化されるオブジェクトの集合体です。
コンストラクタとは、new演算子を使って新たに生成するオブジェクトを初期化するための関数で、オブジェクトのクラスを定義します。

JavaScriptはオブジェクト指向言語と言い、簡単に言うと…
配列aの要素をソートするには、sort()関数に配列aを引数として渡すのではなく、配列aのsort()メソッドを呼び出すのがオブジェクト指向的な考え方です。

// sort()関数に配列aの引数を渡すのがこれ。 sort(a); // オブジェクト指向的な考え方はこれ。sort()メソッドを使います。 a.sort();

変数

変数には宣言する位置によってどこからアクセス出来るかが変わります。
関数の外側で宣言された変数はグローバル変数としてプログラム全体からアクセス可能です。
関数中で宣言された変数は関数スコープとなり、関数中のみアクセス可能です。

var outscope = 'text'; function scope() { // inscopeは関数の外側では呼び出せない。 var inscope = 'text'; }

値

数値

JavaScriptの算術演算子には+-*/%があります。
これらの他にもっと複雑な数学演算を行う関数もあります。以下、例です。

// 2の53乗 Math.pow(2, 53) // -> 1.0 最近傍の整数に変換 Math.round(.6) // -> 1.0 整数に切り上げ Math.ceil(.6) // -> 0.0 整数に切り捨て Math.floor(.6) // -> 5 絶対数 Math.abs(-5) // 引数の中から最大値を返す Math.max(x, y, z) // 引数の中から最小値を返す Math.min(x, y, z) // 0以上1.0未満の擬似乱数を返す Math.random() // π 円周率 Math.PI // 自然対数の底（意味は知らない） Math.E // 3の平方根 Math.sqrt(3) // 3の立方根 Math.pow(3, 1/3) // 三角関数 Math.cos Math.atan もあるよ。 Math.sin(0) // 10の自然対数 Math.log(10) // 底が10の100の対数（100の常用対数） Math.log(100) / Math.LN10 // 底が2の512の対数 Math.log(512) / Math.LN2 // Math.Eの3乗 Math.exp(3)

色々あって、大半のプログラミング言語では小数点を厳密に表現出来ないです。
なので、小数点の計算は整数までスケールを調整して行いましょう。br詳しくは2進浮動小数点と丸め誤差を調べるといいです。
